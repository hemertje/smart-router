### Installation
```bash
# Clone the repository
git clone https://github.com/universal-vibe/smart-router.git

# Install dependencies
cd smart-router
npm install

# Build the extension
npm run compile

# Package for VS Code
vsce package
```

### VS Code Extension
1. Download `smart-router-v2.6.0-complete-roadmap.vsix`
2. Install in VS Code: `Extensions â†’ Install from VSIX`
3. Configure OpenRouter API key in settings
4. Start met `@smart` command

### Agent Marketplace
```typescript
// Create custom agent

  name: 'React Component Generator',
```
